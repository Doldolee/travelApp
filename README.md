# 여행지 상품 추천 어플


### 프로젝트 추진배경
여행지에서 같은 물건을 더 높은 가격을 지불하고 사는 경우가 많습니다. 이에 저희는 여행지에서 적절한 상품을 선택할 수 있도록 정보를 주는 어플을 제작하였습니다.

### 기능
- 여행지 상품 검색 기능
- 여행지 상점 추천 기능
- 상품 북마크 기능
- 상품 검색정보 저장 기능
- 커뮤니티

### 프로젝트 수행 기간
 2021.03. ~ 2021.11

### 기술 스택
- Kotlin
- Android studio
- fireBase
- Spring
